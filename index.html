Safety Compliance

Acerta identifies defects in safety-critical software

Acerta provides new evidence based confirmation measures that can help you identify systemic defects in safety-critical software. The automatically generated insights make standard compliance and certification a simpler, cost effective, and evidence driven process.

How does it work?

Acerta uses logs of system activity to extract key insights about system behavior. These insights are then used as evidence that the system complies with safety requirements. Acerta makes it easy to demonstrate that the system functions correctly during run time, and can automatically produce a -ready-to-use report for certification.

No need for customization

Operating systems, hardware silicon, and runtime environments provide different means to collect logs. Acerta does not impose restrictions on the data collected by the system. It uses any type of log format and adapts it to the type of information that is available. As such, Acerta requires minimal adaptation, allowing you to get quality evidence in no time.
